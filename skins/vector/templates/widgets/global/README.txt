 ==================================================
  GLOBAL WIDGETS DIRECTORY
 ==================================================

 Global widgets are widgets that can be displayed
 in any region of any layout.  Follow the steps
 below to create a global widget:

 1.  Create a php template file in this directory
     with a descriptive name.  Ensure that you
     give it a .php extension.

 2.  Select the widget in the Theme Configuration
     panel for the regions that you want it to
     appear in.

 3.  Save changes in the Theme Configuration
     Panel.
